# Hospital-Emergency-Dashboard--PBI
This project involves the development of a comprehensive Hospital Emergency Dashboard using Power BI to visualize and analyze emergency room performance metrics.
This project involves the development of a comprehensive Hospital Emergency Dashboard using Power BI to visualize and analyze emergency room performance metrics. The dashboard provides actionable insights into patient flow, wait times, and operational efficiency, helping hospital management monitor daily and monthly trends.
Key Features & Metrics:
Operational Overview: Tracks total Number of Patients (431) and Average Wait Time (36.7 Min).
Performance Tracking: Monitors the Patient Satisfaction Score (4.72) and tracks the percentage of patients seen within the 30-minute target (34.34%).
Patient Demographics: Provides breakdowns by Age Group, Gender, and Race to understand the patient base.
Operational Trends: Includes heatmaps/charts for Number of Patients by Day and Time Intervals to identify peak hours and days.
Admission Insights: Analyzes Patient Admission Status (Admitted vs. Not Admitted) and Department-wise Referrals.
